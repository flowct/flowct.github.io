Overview

Infrastructure

FlowCT automatically creates the entire infrastructure in the cloud or private data center just with the push of a button.

CLI & Web UI Access

FlowCT provides a powerful CLI as well as a Web UI based interface for both interacting as well as monitoring.

Log Management

Automatic and complete log management to analyze any test result. Also, provides easy integration with any CI/CD system.



Features

Infrastructure Orchestration

FlowCT creates the entire Docker-based containerized infrastructure for your automated testing from scratch, which can be reused to rerun tests or deleted for a clean system.

Easy YAML based Definition Script

FlowCT infrastructure definition and test scripts are based on industry standard YAML format to make new user oboarding extremely easy.

Cloud Based or On-Premises

FlowCT is architected to meet every need and can be deployed anywhere: Public Cloud, Private Cloud, Hybrid-Cloud or in your private datacenter.

Awesome Support

FlowCT is dedicated to provide best-in-class support and we offer various commercial support service levels and associated SLAs.

Test any Hardware or Software

FlowCT is architected to test any network connected hardware or software system. Some common examples include websites, Routers and Switches, Settop Boxes, Modems and various IoT modules.

Beautiful Management Dashboard and CLI

FlowCT comes with a beautiful and user-friendly management dashboard. We also offer a powerful CLI to allow automation and for those who prefer a CLI interface.

Powerful Test Log Management

FlowCT is pre-integrated with ELK-Stack which comes with powerful log query and management capabilities.

Extensible Architecture

FlowCT is extremely extensible and can be made to integrate in virtually any development, design or build system and can support any current and new software, hardware and protocols.
